# Kavomaz Books Website

Upload these files to your GitHub repository root for the website:

https://books.kavomaz.com

Included files:
- index.html
- life.html
- kids.html
- style.css
- robots.txt
- sitemap.xml
- CNAME
- images folder

Important:
1. Replace the placeholder buttons marked with href="#" with your real Amazon, Gumroad, Etsy, SellApp, or free sample links.
2. Put your book covers inside the images folder using the recommended filenames.
3. In GitHub Pages, set the custom domain to:
   books.kavomaz.com
4. In Cloudflare DNS, create:
   Type: CNAME
   Name: books
   Target: YOUR-GITHUB-USERNAME.github.io
   Proxy status: DNS only
